## Question

<Aside type="note" title="Task Requirements | Kernel and System Info">

Write the Linux Kernel release into `/opt/course/1/kernel`.

Write the current value of Kernel parameter `ip_forward` into `/opt/course/1/ip_forward`.

Write the system `timezone` into `/opt/course/1/timezone`.

ℹ️ If no server is mentioned in the question text, you'll need to create your solution on the default terminal
</Aside>

## Answer

<details>
<summary>Click to see the solution</summary>

<Aside type="tip" title="Solution Commands">

```bash
# display kernel version
➜ uname -r

# output value of kernel parameter
➜ cat /proc/sys/net/ipv4/ip_forward

# get current timezone
➜ date +%Z

cat /etc/timezone # also possible
```

</Aside>

The files should look like:

<Tabs>
<TabItem label="/opt/course/1/kernel">
```bash
5.15.0-69-generic
```

</TabItem>
<TabItem label="/opt/course/1/ip_forward">
```bash
1
```

</TabItem>
<TabItem label="/opt/course/1/timezone">
```bash
UTC
```
</TabItem>
</Tabs>
