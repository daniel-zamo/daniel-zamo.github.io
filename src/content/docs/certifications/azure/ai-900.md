# Microsoft Certified: Azure AI Fundamentals (AI-900)

This certification is the perfect starting point for anyone looking to understand the fundamentals of Artificial Intelligence (AI) and Machine Learning (ML) within the Microsoft Azure ecosystem. It's designed not just for technical professionals, but for anyone—including sales, marketing, and business leaders—who wants to grasp how Azure can be used to build powerful AI solutions.

The exam covers foundational knowledge of AI concepts and how to apply them using Azure services for machine learning, computer vision, natural language processing, and generative AI.
