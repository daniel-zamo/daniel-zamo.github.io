<Aside type="tip" title="tar + bzip2 commands">

Los archivos `.tar.bz2` (o a veces solo `.tbz2` o `.tbz`) son muy comunes en Linux y combinan dos herramientas: `tar` para archivar múltiples archivos en uno solo, y `bzip2` para comprimir ese archivo resultante con una alta relación de compresión.

Aquí te explico cómo comprimir y descomprimir estos ficheros usando el CLI de Linux.
</Aside>

## tar + bzip2

### Conceptos Clave

*   **`tar`**: Una utilidad que se usa para *archivar* (crear un solo archivo a partir de muchos, o extraer archivos de un archivo). Por sí mismo, `tar` no comprime.
*   **`bzip2`**: Una utilidad de compresión que ofrece una mejor compresión que `gzip` (aunque es un poco más lento). Los archivos comprimidos con `bzip2` tienen la extensión `.bz2`.

Cuando ves `.tar.bz2`, significa que primero se usó `tar` para archivar los ficheros y directorios en un archivo `.tar`, y luego ese archivo `.tar` se comprimió con `bzip2`, resultando en `.tar.bz2`.

### Descomprimir un archivo `.tar.bz2`

Para descomprimir y extraer un archivo `.tar.bz2`, usaremos el comando `tar` con las opciones adecuadas.

**Sintaxis básica:**

```bash
tar -xf archivo.tar.bz2
```

Vamos a desglosar las opciones:

*   `-x` (o `--extract`): Indica a `tar` que debe extraer los archivos del archivo.
*   `-f` (o `--file`): Especifica que el siguiente argumento es el nombre del archivo con el que `tar` debe trabajar.
*   `-j` (o `--bzip2`): Esta es la clave para manejar archivos `bzip2`. Le dice a `tar` que el archivo está comprimido con `bzip2` y que debe usar `bzip2` para descomprimirlo antes de extraer. (Alternativamente, `tar` moderno suele detectar automáticamente la compresión por la extensión, pero `-j` es explícito y seguro).

**Ejemplo:**

Supongamos que tienes un archivo llamado `mi_copia_de_seguridad.tar.bz2`.

```bash
tar -xjf mi_copia_de_seguridad.tar.bz2
```

**Opciones adicionales útiles para la descompresión:**

*   `-v` (o `--verbose`): Muestra una lista detallada de los archivos a medida que se extraen. Es muy útil para ver el progreso y qué archivos se están extrayendo.
    ```bash
    tar -xjvf mi_copia_de_seguridad.tar.bz2
    ```
*   `-C /ruta/destino`: Extrae los archivos en un directorio específico en lugar del directorio actual.
