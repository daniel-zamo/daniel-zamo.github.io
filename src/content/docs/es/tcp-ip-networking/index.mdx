## ¿Qué es TCP/IP?

El **Protocolo de Control de Transmisión/Protocolo de Internet (TCP/IP)** es el conjunto de protocolos de comunicación estándar que permite la interconexión de dispositivos en redes locales y en Internet. Este modelo de cuatro capas (Aplicación, Transporte, Internet y Acceso a la Red) proporciona la base para prácticamente todas las comunicaciones de red modernas.

### Conceptos clave que exploraremos:

- **Direccionamiento IP**: Asignación y gestión de direcciones IPv4 en diferentes segmentos de red
- **Subnetting**: Técnicas para dividir redes grandes en subredes más pequeñas y manejables  
- **Routing**: Configuración de rutas entre diferentes segmentos de red
- **Switching**: Operación de switches en redes de área local (LAN)
- **Protocolos de red**: DHCP, DNS, ARP y otros servicios esenciales de red
