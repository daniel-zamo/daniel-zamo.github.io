## Question

You're required to perform changes on LVM volumes:

1. Reduce the volume group vol1 by removing disk /dev/vdh from it
1. Create a new volume group named vol2 which uses disk /dev/vdh
1. Create a 50M logical volume named p1 for volume group vol2
1. Format that new logical volume with ext4
