Debian 12 - configuración inicial

Primera configuración luego de instalar Debian 12

07 July 2023

En esta sección se realizan tareas de configuración inicial de una distribución Linux Debian 12 (Bookworm).

Nota del autor: Las tareas realizadas en este apartado, pueden ser ejecutadas en cualquier distribución derivada de la misma versión Debian Bookworm (Ejemplo: Linux Q4Os, Crunchbangplusplus, Linux MX, etc).

Configurando usuario sudo

El siguiente comando configura el usuario de login $USER para que ejecute los comandos con privilegios de root, mediante el uso de sudo sin que se le pida contraseña.

Usuario para comando sudo (sin password)

El siguiente comando configura el usuario de login $USER para que ejecute los comandos con privilegios de root, mediante el uso de sudo sin que se le pida contraseña.

[dzamo@victus beta02]$ echo "$USER ALL=(ALL) NOPASSWD:ALL" | sudo tee /etc/sudoers.d/$USER

Habilitar modulo wheel de pam para el grupo adm

La siguiente configuración, habilita el módulo pam su wheel, para limitar solo a los usuarios del grupo adm utilizar el comando su (escalar a root mediante su quedará permitido para los usuarios del grupo wheel).

[dzamo@victus beta02]$ # Usuarios grupo adm en /etc/pam.d/su
[dzamo@victus beta02]$ sudo sed -i "s/^# \{1\}auth \{7\}required \{3\}pam_wheel.so$/&\nauth    required  pam_wheel.so  group=adm/" \
    /etc/pam.d/su
[dzamo@victus beta02]$ sudo usermod -G adm -a $USER

Instalación de algunos paquetes

[dzamo@victus beta02]$ sudo apt -y install tmux  rsync neovim neofetch btop htop vim \
    vim-scripts exuberant-ctags libtemplate-perl \
    libtemplate-plugin-gd-perl libtemplate-plugin-xml-perl fonts-dustin libtest-fatal-perl

Instalar/configuración inicial de Python

[dzamo@victus beta02]$ sudo apt install python3-venv pip python3-pip
[dzamo@victus beta02]$ sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.11 1
