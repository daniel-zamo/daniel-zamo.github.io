02 - Fedora 38 - Instalación del sistema (mínima)

Esta página explica la instalación de Fedora 38 (link a sitio externo) - minimal

02 July 2023

En las siguientes capturas se muestran algunas pantallas (mas relevantes) del asistente de instalación del sistema, resaltando algunos puntos a considerar.

Install Fedora 38 & Welcome to Fedora 38

En la selección de paquetes de dicho asistente, solo se selecciona el metapaquete Minimal, de modo de mantener al mínimo los paquetes instalados. De esta manera, luego podremos ampliar el sistema (según los paquetes que se agreguen) para desempeñarse como servidor o como estación de trabajo (Desktop) con algún gestor de ventanas (como Gnome o KDE).

Nota del autor: En esta entrada, no son mostradas todas las pantallas del asistente de instalación, solo las que se consideran mas relevantes de referir.

Manual partitioning

En esta instalación se utilizará arranque UEFI. En las capturas siguientes se muestran el particionado manual realizado con el propio asistente.

El tamaño del disco es 20Gb, y el esquema de particionado del filesystem requerido es:

punto montaje tamaño tipo
/ 14.66G VG - LV - ext4
/boot/efi 600 Mb EFI
/boot 768 Mb ext4
swap 4 G swap
