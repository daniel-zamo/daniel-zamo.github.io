CLI en Linux

d@Zamo enero 30, 2025 #Linux #CLI #sistemas-operativos

En sistemas Linux y Unix, la CLI (Command Line Interface) es la herramienta principal para interactuar con el sistema a través de comandos escritos. A diferencia de un entorno gráfico (GUI), la CLI permite un control más preciso y eficiente del sistema, especialmente útil para administración, automatización y scripting, principalmente en sistemas operativos orientados al comando.

¿Qué es un Shell en Linux/Unix?

Un shell es un programa que actúa como intermediario entre el usuario y el sistema operativo. Su función es interpretar los comandos que ingresamos y ejecutarlos.

Existen varios tipos de shells en Linux/Unix, pero uno de los más comunes y ampliamente utilizados es Bash (Bourne Again Shell).

¿Qué es Bash y por qué es importante?

Bash es un shell muy popular en sistemas Linux y macOS, y es el sucesor mejorado del shell original de Unix, el Bourne Shell (sh).

✔️ Se usa para ejecutar comandos interactivos.

✔️ Permite la automatización mediante scripts (.sh).

✔️ Soporta variables, funciones y estructuras de control (if, for, while).

✔️ Administra procesos, redirección de entrada/salida y permisos.

Ejemplo de un comando básico en Bash:

ls -l /home

El comando anterior lista los archivos y directorios en /home con detalles extendidos.
