Servidor NFS Debian

En esta entrada se instala un servidor NFS sobre un sistema Debian. Esta misma configuración es funcional para cualquier derivado Linux de esta distribución, por ejemplo Ubuntu LTS o Linux Mint. Especificación Nombre de dominio: my.domain.lan IP LAN: <MY_LAN_IP/MASK> (CIDR IPv4 LAN) Directorio a compartir: /home/nfs-share Instalación servidor NFS sudo apt -y update sudo apt -y install nfs-kernel-server Configuración inicial/básica sudo sed -i.org \ -e 's/^# Domain = localdomain$/&\nDomain = my.domain.lan/' \ /etc/idmapd.conf sudo mkdir -pv /home/nfs-share echo '/home/nfs-share <MY_LAN_IP/MASK>(rw,no_root_squash)' \ | sudo tee /etc/exports sudo systemctl restart nfs-server Algunos comandos de verificación (simple) sudo exportfs sudo systemctl status nfs-server
