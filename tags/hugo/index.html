Archetypes en Hugo

Hugo incluye de forma nativa la funcionalidad de poder predefinir el contenido inicial (incluido el front matter) de las entradas que se crean con el comando hugo new este tipo de elementos son los templates Archetypes del generador de contenido estático Hugo. Configuración de una plantilla Archetypes Crear un archivo de arquetipo personalizado En el proyecto Hugo en que se este trabajando, este tipo de plantillas se ubican en la carpeta ROOT_SITE/archetypes. En este ejemplo, y como este sitio que se esta desarrollando utiliza el Theme Toha es que la mayoría del contenido son o se ubican en ROOT_SITE/content/posts es que se crea la plantilla ROOT_SITE/archetype/posts.md.
