Instalación de sis. op. desde red utilizando iVentoy

  

  • Linux
  • Servicios
  • Protocolos
  • PXE-boot
  • ISO

En esta entrada se instala dos sistemas operativos mostrando la utilización de iVentoy, el cual utiliza el entorno PXE de TCP/IP.

En este caso se utilizará un Desktop/CLI basado en Linux, pero la aplicación al estar basada en TCP/IP también esta implementada para otros sistemas operativos como Windows.
