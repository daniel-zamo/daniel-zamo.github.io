Instalando Python sobre Ubuntu 22.04 o superior

En esta publicación se instala y configura el interprete Python en versión 3.10. sobre una distribución Linux basada en Ubuntu 22.04 (aunque en superior y/o derivada, o el mismo Debian (distribución en la que se basa Ubuntu) también es funcional).

Tarea previa

Actualizar sistema

sudo apt update
sudo apt -y upgrade

Instalación de Python

sudo apt install python3.10 python3.10-venv python3-pip

Instalación de paquetes (sugeridos - no obligatorios)

Los siguientes paquetes son opcionales y NO son directamente necesarios para el interprete Python.

Instalo otros paquetes (compiladores, desarrollo) extras

sudo apt install gcc-11-locales debian-keyring g++-multilib g++-11-multilib \
  gcc-multilib autoconf automake libtool flex bison gdb  gcc-11-multilib bzr \
  libgd-tools build-essential

Configuración path Python (por defecto)

Configuro path defecto para python

which python
ll /usr/bin/python3*
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 1
which python
python -V
